# MySims Manual
Manual implementation for MySims, compatible with both PC/Switch and Wii versions.

## What is randomized
Items you will receive: Sims, Essences, Blueprints, progression Tools (Crowbar, Saw, Pickaxe, Blowtorch).
Checks: Found Essences, completed tasks, Sims' best friend rewards (if enabled in the options).

## Goal of the game
The goal is to reach the Star Level of your choosing by gathering points gained from completing Commercial Sims' tasks. The default Level goal is 5, you can also set it to 3 or 4.
